Монолит - подход к разработке, когда весь код находится в одном проекте. Допустим, у вас есть интернет магазин по продаже яблок. У вас есть авторизация, каталог, оплата, корзина, уведомления и все они написаны только на одном языке. Если произойдет какая-то критическая ошибка в одном из этих сервисов, у вас может упасть всё приложение. Также, если вы захотите добавить новый функционал, вам нужно посмотреть не ломает ли он что-то старое, каждый раз у вас будет уходить много времени на сборку проекта, поэтому ваша эффективность падает. Звучит это все очень плохо, особенно, если у вас большой бизнес.

Поэтому появились микросервисы. Микросервисы - подход, когда ваш код разбит на разные части, каждая из которых выполняет свой функционал и изолирована. Ваш интернет магазин будет состоять из различных микросервисов, указанных выше, которые могут быть написаны на разных языках. Теперь, если у вас что-то сломается в одной из частей проекта, все остальное из-за этого не пострадает, также внеся изменения в один из сервисов, они не сломают другие части проекта. Сборка также станет быстрее. При росте нагрузки вы можете увеличивать количество только определенных сервисов, что экономит ресурсы.

Языки программирования

Go (часто также golang) — компилируемый многопоточный язык программирования, разработанный внутри компании Google. Поддерживает объектно-ориентированный, и функциональный стили.

Python — мультипарадигмальный высокоуровневый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью, ориентированный на повышение производительности разработчика, читаемости кода и его качества, а также на обеспечение переносимости написанных на нём программ

Java — строго типизированный объектно-ориентированный язык программирования общего назначения, разработанный компанией Sun Microsystems (в последующем приобретённой компанией Oracle). Разработка ведётся сообществом, организованным через Java Community Process; язык и основные реализующие его технологии распространяются по лицензии GPL. Права на торговую марку принадлежат корпорации Oracle.
